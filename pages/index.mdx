## The Nystron Solar Web Project ‚òÄÔ∏è

[This project](https://solar.nystron.com.br) represents the fusion of cutting-edge technology, renewable energy, and precise financial handling, providing Nystron Solar clients with a seamless experience for monitoring their solar generation data.

### Key Features

- Utilizes Symfony framework and a suite of Symfony packages for robust functionality.
- Multilingual support with Symfony's translation capabilities.
- Secure user authentication and access control using Symfony's security components.
- Efficient messaging, forms, and email functionality for seamless client communication.
- A user-friendly dashboard that displays solar energy generation data.
- Integration with Solar Api's for accurate solar inverter data retrieval.
- Bill analysis and cost tracking for each client.

### Precise Money

[This package](https://github.com/TheDevick/PreciseMoney) is a powerful tool for handling monetary values with precision in PHP applications by working with decimal values down to cents. Regular updates and bug fixes ensure that the package remains reliable and up-to-date.

### _Nystron Solar_ - Electric Bill Extractor

[This package](https://github.com/NystronSolar/ElectricBillExtractor) simplifies the process of parsing and retrieving information from PDF bills, making it a valuable asset for developers and businesses dealing with financial data.

### _Nystron Solar_ - Growatt Api

[This package](https://github.com/NystronSolar/GrowattApi) is a powerful tool for integrating and consuming data from the Growatt solar inverter monitoring API. It simplifies the process of fetching, analyzing, and utilizing solar energy data.
